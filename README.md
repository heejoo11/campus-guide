# 캠퍼스 가이드 (Campus Guide)

> "읽는 정보에서 바로 움직이는 정보로"
>
> 파편화된 대학 내 공지사항을 수집·구조화하여 학생 개개인의 맥락에 맞는 맞춤형 캠퍼스 가이드 서비스입니다.
>
> ---
>
> ## 프로젝트 소개
>
> 국민대학교 학생들이 학교 앱의 수많은 알림 속에서 **본인에게 필요한 공지만 선택적으로 받을 수 있도록** 돕는 서비스입니다.
>
> ### 핵심 기능
> - 학교 메인 및 학과 게시판 공지 자동 수집
> - - 학년·학과·키워드 기반 개인화 필터링
>   - - AI 마감일·대상 자동 추출 및 알림
>     - - 챗봇 오버레이를 통한 즉답형 Q&A
>      
>       - ---
>
> ## 팀 구성
>
> | 팀 | 역할 |
> |---|---|
> | 백엔드 | 크롤러 개발, 데이터 정규화, REST API |
> | 프론트엔드 | 웹 페이지 UI, 챗봇 오버레이 |
> | AI | 공지 분석, 마감일 추출, RAG Q&A |
>
> ---
>
> ## 기술 스택
>
> ### 프론트엔드
> - React + React Router v6
> - - TanStack Query (React Query)
>   - - Axios + Tailwind CSS
>
